# Waste to Energy Project

## 📌 Overview
This project demonstrates how waste materials can be converted into useful energy.

## ⚙️ Working Principle
The waste is processed in a chamber where heat energy is generated. This energy is then converted into electrical energy.

## 🔧 Components Used
- PVC pipes
- Heating chamber
- Wires
- LED bulb
- Power supply
- Circuit board

## 🛠️ Process
1. Waste is collected and placed in chamber
2. Heat is generated
3. Energy conversion process starts
4. Electricity is produced (LED glows)
